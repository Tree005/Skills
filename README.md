## 🔧 使用方式

1. 选择你需要的 Skill，进入对应文件夹
2. 按各 Skill 的 README 说明安装到对应的 Agent skills 目录下（如 WorkBuddy 的 `~/.workbuddy/skills/`）
3. 在对应的 Agent 中即可使用
